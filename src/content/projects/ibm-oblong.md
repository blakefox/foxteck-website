Assisted with the installation and creation of content for the IBM Austin immersion room — a massive multi-display environment built on Oblong's gestural computing platform.

Worked directly with the Oblong team to build shader content and Web 3D content within the core mechanics of the system. Led R&D on custom software deployment across the server stack to enable dynamic 3D visuals that spanned the full screen array — creating an environment where data, AI output, and visual content could coexist at an unprecedented scale.

The immersion rooms represented a new paradigm for enterprise collaboration and AI demonstration, allowing teams to interact with Watson-powered data in a physically immersive space.
